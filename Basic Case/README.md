# FPGA PUF (物理不可克隆函数) 实现

## 📌 项目概述

本项目在 FPGA 上实现了一个基于环形振荡器的物理不可克隆函数，能够根据芯片制造差异生成独一无二的硬件指纹。

## 🛠 核心功能

- **挑战-响应机制**：支持 384 位挑战输入，产生 96 位响应
- **多时钟域采样**：使用 PLL 生成 4 个时钟，增强熵源
- **5次采样多数表决**：提高响应稳定性，抑制噪声
- **UART 通信**：115200 波特率，用于接收挑战和发送响应
- **六位数码管显示**：支持按键翻页查看响应数据的不同部分

## 📁 文件说明

- `puf_top.v` - 顶层模块，集成控制和通信，类似于编程里面的main
- `fpga_puf.v` - PUF 核心实现，包含状态机和单元阵列
- `uart_rx.v`/`uart_tx.v` - UART 收发模块
- `majority_vote_96_7.v` - 多数表决模块
- `seg7_driver6.v` - 数码管驱动模块
- `ip_pll.v` - PLL 时钟生成模块

## ⚙️ 使用说明

1. 通过 UART 发送 48 字节挑战数据（384 位）
2. 系统自动进行 5 次采样和多数表决
3. 结果通过 UART 返回（12 字节）
4. 使用板载按键查看响应数据的不同部分

## 🔧 关键参数

- 系统时钟：50MHz
- 串口波特率：115200
- PUF 响应位宽：96 位
- 挑战位宽：384 位
- 采样次数：5 次

## ✨ 特点

- 利用 FPGA 固有偏差，无需额外存储
- 抗噪设计，响应稳定
- 实时显示，便于调试验证
- 标准串口通信，易于集成
