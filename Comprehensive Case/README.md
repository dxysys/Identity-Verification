# 空地协同认证 实现

## 📌 项目概述

本项目搭建实际的空地系统场景，进行空地协同身份认证。

## 🛠 核心功能

- **挑战-响应机制**：发送 384 位挑战输入，产生 96 位响应
- **空地协同认证**：实现低空设备和地面端进行安全快速的身份认证

## 📁 文件说明

- `PUF.py` - 地面端和开发板进行通信

## 🔧 关键参数

- 双方IP地址

## ✨ 特点

- 利用硬件身份认证和智能身份认证，实现空地协同认证
